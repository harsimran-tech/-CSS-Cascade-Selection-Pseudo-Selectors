# 📝 To-Do List – CSS Cascade, Selection & Pseudo Selectors

## Overview

This project is a styled to-do list built as part of **HW #3** for the MMED-1051 course. The focus of the assignment is to apply CSS concepts such as the cascade, generic selectors, ID/class selectors, and pseudo-selectors like `::before` and `::after`.

---

## Features

- Background image with a custom “to-do” theme
- Creative header styled using an ID selector
- Custom fonts and color scheme using Google Fonts
- Checkboxes added to both “In Progress” and “Done” items
- CSS selectors dynamically update the style of list items based on checkbox state
- `::after` pseudo-selector adds fun graphic elements (e.g., ✅ or 🎉) to indicate completion
